# Description

Simple form to collect a company's financial details with the following rules:
<ul>
  <li>The dropdown "Currency base currency" must contain the supported currencies based on the "registered country" selected earlier in the process</li>
  <li>The currency symbol in the "Anual Revenue" fields must show the currency symbol of the currency selected in the "Company base currency"</li>
  <li>The anual revenue and anual revenue on other marketplaces only are required fields</li>
  <li>The anual revenue must be larger than zero</li>
  <li>The anual revenue on other marketplaces only must be smaller that the anual revenue</li>
</ul>
